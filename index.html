Welcome to WiseVoter Pages.

This automatic page generator is the easiest way to create beautiful pages for all of your projects. Author your page content here using GitHub Flavored Markdown, select a template crafted by a designer, and publish. After your page is generated, you can check out the new branch:

  • View Politician (e.g Rahul Gandhi) Bio's
  • Browse constituencies (Consituencies Page)
  • Learn about political parties and their agenda (Parties Page)
  • Visualize data on politicians (Google Fusion Tables - compare politicians, parties etc.)
  • Learn relevant electoral issues (Issues list)
  • Contribute to refine politician bios
  • Towards election date: register to vote, report voting fraud etc.

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
